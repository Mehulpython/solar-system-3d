# 🌍 Interactive 3D Solar System

Real-time 3D solar system explorer built with Three.js. Runs in any modern browser — no install needed.

## Features
- All 8 planets + Sun with realistic procedural textures
- Real-time orbital motion and axial rotation
- Click any planet to fly to it and follow it
- Saturn's rings, Earth's moon and atmosphere
- Speed control (pause to 5x), orbit lines, labels
- Info panel with real astronomical data
- 12,000+ star background

## Tech
- Three.js (WebGL)
- Procedural canvas textures (no external image dependencies)
- Single HTML file — zero build step

## Run
```bash
python3 -m http.server 8090
# Open http://localhost:8090
```

Live at: https://planetarium.mehulai.com
